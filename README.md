# Providing Token

The Token and the Guild (aka the discord server name) have to be provided over a file called `.env` in the same directory as the bot.

Its contents have to be:

`DISCORD_TOKEN=TOKEN` 

`DISCORD_GUILD=GUILD_NAME`
